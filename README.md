# GitHub Organization CLI

## Installation

### npm

```shell
pnpm add --global github-orgs-cli
```

### Locally install by Git

```bash
git clone https://github.com/pers0n4/github-orgs-cli.git
cd github-orgs-cli

pnpm link --global
```

## Usage

```shell
github-orgs-cli <filename>
```
